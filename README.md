# SeaMonkey Expeditions

A responsive travel website showcasing wildlife experiences in Jordan, built with HTML, CSS, and Bootstrap.

## Description

SeaMonkey Expeditions is Jordan's premier wildlife experience company. This website provides information about various expeditions including hiking, camping, sea safaris, and vehicle safaris. Users can explore the diverse wildlife of Jordan, including the Arabian Oryx, birds, water buffalos, and marine life.

The site features a modern, responsive design that works seamlessly across desktop and mobile devices.

## Features

- **Responsive Design**: Optimized for all screen sizes using Bootstrap and custom CSS
- **Hero Section**: Eye-catching landing area with call-to-action
- **About Section**: Information about the company and its mission
- **Expeditions**: Showcase of different trip types with filtering options
- **Gallery**: Visual showcase of captured wildlife and adventures
- **FAQ**: Frequently asked questions about wildlife spotting and travel
- **Contact Form**: Easy way for visitors to get in touch
- **Social Media Integration**: Links to Facebook, YouTube, Instagram, and LinkedIn

## Technologies Used

- **HTML5**: Structure and content
- **CSS3**: Custom styling and responsive design
- **Bootstrap 5.3.7**: Framework for responsive layout and components
- **JavaScript**: Bootstrap's JavaScript for interactive components (accordion, navbar)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/SeaMonkey-Expeditions.git
   ```

2. Navigate to the project directory:
   ```bash
   cd SeaMonkey-Expeditions
   ```

3. Open `index.html` in your web browser to view the site.

## Usage

Simply open the `index.html` file in any modern web browser. The site is fully static and requires no server-side setup.
